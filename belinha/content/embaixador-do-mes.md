## O QUE É O EMBAIXADOR DO MÊS

Todo mês, **o cliente que mais indicações bem-sucedidas gerou** (amigo veio + comprou) vira o **Embaixador MilkyPot do Mês**.

Ele recebe:
- **+5 carimbos** no cartão fidelidade (bônus imediato)
- **Destaque no Instagram** do MilkyPot (@milkypotbr — story + post carrossel)
- **Título** "Embaixador(a) da Ovelhinha 🐑" até o fim do mês seguinte
- **Acesso antecipado** ao próximo produto sazonal (degustação grátis antes do lançamento)

Custo para a loja: **zero** (exceto os carimbos, que já seriam do fidelidade). Retorno: embaixador indica ainda mais agressivamente + UGC orgânico.

## COMO IDENTIFICAR O EMBAIXADOR

### Passo 1 — Contagem na planilha (última semana do mês)

Use a planilha de rastreamento do Indica & Ganha (`whatsapp-indicacao.md`):

```
DATA | QUEM INDICOU | QUEM VEIO | CARIMBO DADO?
```

Conta: quem tem **mais linhas com "Carimbo dado = ✅"** = Embaixador do Mês.

**Empate:** em caso de empate, o prêmio vai para quem indicou **primeiro no mês** (data mais antiga).

### Passo 2 — Confirmação mínima

Só anunciar Embaixador se houver **pelo menos 1 indicação bem-sucedida no mês**.  
Se ninguém indicou: não anunciar, e usar o Template de Ativação abaixo para impulsionar julho.

## BENEFÍCIOS ESTRATÉGICOS

**Para o operador:**
- Reconhecimento zero custo → aumenta qualidade das indicações orgânicas
- UGC (foto do embaixador) = conteúdo autêntico para o Instagram sem custo de produção
- Lista de transmissão WA cresce cada vez que novo embaixador compartilha o anúncio

**Para o cliente embaixador:**
- Status social ("fui o embaixador!") → compartilha espontaneamente
- Degustação VIP = experiência exclusiva → fideliza a nível emocional
- +5 carimbos = metade do caminho para o Mini grátis

**Para clientes gerais:**
- Saber que o prêmio é real (foto publicada, nome citado) → motivação para indicar
- Ciclo virtuoso: indicação → compra → embaixador → mais indicações
