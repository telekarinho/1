## O QUE É ESTA MECÂNICA

**Milestone permanente:** quando um cliente acumula **3 indicações bem-sucedidas** (amigo veio + comprou), ele ganha automaticamente **1 Potinho Grande grátis** — sem precisar ganhar o concurso mensal, sem data de corte.

### Por que isso existe?

| Programa existente | Recompensa | Gatilho |
|-------------------|-----------|---------|
| Indica & Ganha | +1 carimbo | 1 indicação |
| **Milestone 3 indicações** ← este doc | **Potinho Grande grátis** | **3 indicações acumuladas** |
| Embaixador do Mês | +5 carimbos + destaque IG | Mais indicações do mês |

A lacuna: clientes que chegam a 2 indicações não têm motivo concreto para tentar a 3ª. Este milestone resolve isso com uma recompensa imediata e tangível (valor ~R$25–35) que funciona como "prêmio de diplomação" — o cliente vira **Embaixador Oficial** a partir daí.

## COMO FUNCIONA — PASSO A PASSO

### Para o operador

```
1. Registrar TODA indicação bem-sucedida na planilha (ver seção Rastreamento)
2. Quando cliente atingir 3 indicações → enviar Template A (parabéns + Potinho Grande)
3. Cliente aparece na loja → entregar Potinho Grande + foto opcional para IG
4. Atualizar planilha: coluna "Milestone atingido?" = ✅ + data
5. CONTINUAR contando indicações do cliente para o concurso mensal (não zera)
```

### Para o cliente

```
1. Indica amigo (com card de indicação enviado pelo operador — ver whatsapp-indicacao.md)
2. Amigo vem à loja e menciona o nome de quem indicou
3. Cliente acumula: 1ª → carimbo extra | 2ª → carimbo extra | 3ª → carimbo extra + POTINHO GRANDE GRÁTIS
4. A partir da 3ª, vira "Embaixador Oficial" — badge visual no WA + tratamento diferenciado
```

## INTEGRAÇÃO COM OUTROS PROGRAMAS

```
Indica & Ganha (whatsapp-indicacao.md)
       ↓ cada indicação = +1 carimbo
  ESTE PROGRAMA (milestone 3 indicações)
       ↓ ao atingir 3 = Potinho Grande grátis + badge Embaixador
  Embaixador do Mês (embaixador-do-mes.md)
       ↓ quem mais indicou no mês = +5 carimbos + destaque IG + degustação VIP
```

**Pontos de atenção:**
- Contagem de indicações NÃO zera após o milestone — continua para o concurso mensal
- Um cliente pode ganhar o milestone E o Embaixador do Mês no mesmo período
- O milestone é PERMANENTE: cada cliente só ganha 1 vez (verificar coluna "Potinho dado?")
- Se cliente perguntar se pode ganhar de novo: "Esse prêmio é uma vez só — mas você concorre todo mês ao Embaixador do Mês 🏅"

## FAQ DO OPERADOR

**"E se o cliente pedir pra indicar o próprio filho/parente que mora na mesma casa?"**  
→ Não vale. Regra já está no `whatsapp-indicacao.md`: indicação deve ser de pessoa diferente do grupo familiar imediato.

**"O cliente pode indicar por WhatsApp sem vir à loja?"**  
→ Sim. O amigo pode vir à loja mencionando o nome de quem indicou, ou pode pedir pelo WhatsApp da loja mencionando quem indicou. O que vale é a compra confirmada + nome do indicador registrado.

**"Posso oferecer este benefício como promoção de inauguração tardia?"**  
→ Sim. Use o Template PROG-1 nos primeiros 30 dias para clientes da inauguração que ainda não indicaram ninguém — é uma ótima reativação pós-abertura.

**"O que fazer se a planilha foi perdida e o cliente jura que tem 3 indicações?"**  
→ Favor da dúvida UMA VEZ. Anotar no obs: "concedido por boa-fé [data]". Alertar gerente.
