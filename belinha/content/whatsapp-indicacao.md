# Programa de Indicação MilkyPot — Templates WhatsApp

> **Para:** Operador/Atendente da loja Muffato Londrina
> **Canal:** WhatsApp (43) 99804-2424
> **Criado por:** Belinha — Ciclo #41 (2026-04-26)
> **Mecânica:** cliente indica 1 amigo que compra → ganha 1 carimbo extra no cartão fidelidade

---

## COMO FUNCIONA (resumo para o operador)

1. **Operador envia** o Template A para clientes que já compraram (1x ou mais)
2. **Cliente encaminha** o Template B para um amigo
3. **Amigo vem à loja** e menciona o nome de quem indicou (ou mostra o card)
4. **Operador registra**: carimbo extra para quem indicou + carimbo normal para o novo cliente
5. **Limite:** 3 indicações por semana por cliente (evita gamificação excessiva)

> 💡 **Dica operador:** anote em papel/planilha: "Cliente X indicou Cliente Y — data — carimbo extra dado". Simples, sem sistema.

## TEMPLATE C — Confirmação para quem indicou (após amigo comprar)

> Operador envia para o cliente original DEPOIS que o amigo visitou e comprou.

```
Oi [NOME]! 🐑🎉

*Boa notícia:* [NOME DO AMIGO] passou aqui no MilkyPot e adorou!

Como combinado, adicionamos *+1 carimbo* no seu cartão fidelidade! 🏅

Seu cartão agora:
🐑 [N] carimbos — faltam [10-N] pra ganhar o Mini grátis!

Quer ver seu cartão? 👇
milkypot.com/cartao-fidelidade.html?stamps=[N]

Valeu por espalhar a felicidade! 🌈✨
```

> **Como preencher:**
> - `[N]` = número atual de carimbos do cliente (depois de somar o extra)
> - `[10-N]` = 10 menos N
> - Link funciona direto — sem login, sem app
