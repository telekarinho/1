## STORY 2/2 — RESPOSTA + CTA

**Quando:** Qui 11/06 · 20h30 (imediatamente após a story 1)

**Fundo/visual:**
- Fundo branco ou creme claro — contraste limpo após o vinho
- Foto ou flatlay: **dois potinhos separados**, etiquetas com nomes visíveis (ex: "PEDRO ❤️" e "LARA ❤️")
- Iluminação quente, produto em destaque
- Ovelhinha pequena entre os dois potinhos (no centro, abaixo das etiquetas)

**Texto na tela (topo):**
```
No nosso, não precisa dividir.

Cada um com o nome de vocês
na etiqueta.
```

**CTA (rodapé, destaque):**
```
Amanhã · Sexta 12/06 · 14h–23h
📍 Muffato Londrina

→ Manda  NAMORADOS26  no WhatsApp
```

**Sticker:**
- Link sticker: "💕 Quero o meu potinho" → `https://wa.me/5543998042424`

**Nota operacional:**
- A etiqueta com o nome é o maior diferencial visual — se possível, preparar os dois potinhos com etiquetas antes de fotografar

- Fundo branco/creme + iluminação quente = foto bonita sem câmera profissional (iPhone + janela com luz natural)
