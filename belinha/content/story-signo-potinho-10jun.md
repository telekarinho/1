**Quando:** Seg 10/06 · 18h00

**Fundo/visual:**
- Fundo azul noite pastel (azul lavanda escuro, não preto) com estrelinhas espalhadas
- Ovelhinha no centro olhando para cima (como procurando estrelas), pequena luneta ou expressão curiosa
- Título em fonte bold branca no topo, bem espaçado

**Texto na tela:**
```

qual é o seu potinho. ✨🐑

Qual é o seu signo?
```

**Sticker interativo:**
- Poll sticker (votação) com duas opções populares pra volume de resposta:
  - `♊ Gêmeos ou Cancer`

- Posição: centro-baixo, bem visível
- Alternativa: Question sticker "Manda seu signo aqui 👇" — mais livre mas menos conversão

- 5s de duração — texto curto para leitura fácil

- Fundo azul noite + estrelinhas cria visual diferente de tudo na timeline (que costuma ser rosa/branco para Namorados)
- Ovelhinha olhando para as estrelas = personagem narrativo que "consulta" o horóscopo
