## INSTRUÇÕES PARA IMPRESSÃO

### Opção A — Papel Fácil / gráfica rápida Londrina (R$3–8)

1. Salvar o texto em Google Docs com a formatação final
2. Inserir QR code gerado em `qr-code-generator.org` → link: `https://wa.me/5543998042424`
3. Imprimir em papel couchê 150g (não borrar com umidade do balcão)
4. Plastificar (R$2 extra) — imprescindível para durar semanas

### Opção B — Canva (grátis, resultado visual melhor)

1. Criar conta em canva.com → buscar template "Table tent" ou "Plaquinha balcão"
2. Usar dimensão A5 (148 × 210mm) ou A6 (105 × 148mm)
3. Cores: roxo `#5B2D8E`, rosa `#F48FB1`, branco `#FFFFFF`
4. Fonte: Nunito ou Poppins (similar à identidade MilkyPot)
5. Baixar em PDF alta resolução → imprimir em gráfica local

### Opção C — Print rápido em casa (mínimo viável)

1. Copiar o texto da "Versão Compacta A6" acima
2. Colar em Word/Google Docs, ajustar para cartão 10×7cm
3. Imprimir em impressora doméstica, recortar, plastificar com fita adesiva larga
4. Duração estimada: 1–2 semanas antes de deteriorar

## POR QUE PLAQUINHA FUNCIONA MELHOR QUE SÓ O SCRIPT VERBAL

| Só script verbal | Plaquinha + script |
|-----------------|-------------------|
| Depende do atendente lembrar | Sempre visível, 100% do tempo |
| Varia por turno / pessoa | Consistente em qualquer turno |
| Cliente pode esquecer depois | Cliente relê antes de ir embora |
| Zero registro visual | Reforça identidade visual da marca |
| Não funciona no delivery | Card na sacola = funciona em qualquer canal |

**Princípio:** A plaquinha é o "lembrete silencioso" — trabalha enquanto o atendente está ocupado.
