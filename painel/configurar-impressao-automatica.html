🖨️ Impressão Automática (sem clicar)

Configure o Chrome do PDV para imprimir o recibo direto na impressora sem mostrar a janela de "Imprimir".

⚡ Resultado: Depois desse setup, ao clicar em Fechar no PDV o recibo sai direto na impressora térmica — sem dialog, sem clique extra. A atendente não para o atendimento.

📋 Passo 1: Definir a impressora térmica como padrão do Windows

  1. Abra Configurações do Windows → Bluetooth e dispositivos → Impressoras e scanners
  2. Procure sua impressora térmica (ex.: Elgin i9, Bematech MP-4200, Epson TM-T20)
  3. Clique nela → Definir como impressora padrão
  4. Imprima uma página de teste pra confirmar que tá imprimindo certo
✅ Importante: Desmarque a opção "Permitir que o Windows gerencie minha impressora padrão" no Windows 11, senão ele troca sozinho.

🚀 Passo 2: Abrir o Chrome com a flag --kiosk-printing

Essa flag faz o Chrome imprimir silenciosamente na impressora padrão sem mostrar janela.

2.1. Clique com o botão direito no atalho do Chrome (área de trabalho ou barra de tarefas) → Propriedades

2.2. No campo Destino, adicione --kiosk-printing no final (depois das aspas e com um espaço):

"C:\Program Files\Google\Chrome\Application\chrome.exe" --kiosk-printing

2.3. Clique em Aplicar → OK

2.4. Feche TODAS as janelas do Chrome abertas (importante!)

2.5. Abra o Chrome pelo atalho modificado

💡 Dica: Crie um atalho dedicado "MilkyPot PDV" que já abre direto em milkypot.com/painel/pdv.html:
"C:\Program Files\Google\Chrome\Application\chrome.exe" --kiosk-printing --app=https://milkypot.com/painel/pdv.html

No Terminal:

open -a "Google Chrome" --args --kiosk-printing

Pra criar um atalho permanente, salve isso como milkypot-pdv.command e dê permissão de execução:

#!/bin/bash open -a "Google Chrome" --args --kiosk-printing --app=https://milkypot.com/painel/pdv.html
google-chrome --kiosk-printing --app=https://milkypot.com/painel/pdv.html

🧪 Passo 3: Testar

  1. Abra o PDV pelo atalho modificado
  2. Faça uma venda de teste (R$ 1,00 por exemplo)
  3. Clique em Fechar Pedido
  4. O recibo deve sair direto na impressora sem mostrar a janela "Imprimir" 🎉
✅ Funcionou? Pronto, agora toda venda imprime automaticamente. A atendente só precisa clicar em "Fechar Pedido" uma vez.
❌ Não funcionou? Verifique:
  • Você fechou TODAS as janelas do Chrome antes de reabrir?
  • A flag tá no destino do atalho (não no Chrome aberto)?
  • Impressora térmica tá definida como padrão do Windows?
  • Bloqueador de popup do Chrome permite milkypot.com?

🛡️ Passo 4 (opcional): Modo Kiosk Completo

Pra deixar o PDV em tela cheia sem bordas do Chrome (parece um caixa de verdade):

"C:\Program Files\Google\Chrome\Application\chrome.exe" --kiosk --kiosk-printing https://milkypot.com/painel/pdv.html

Pra sair do modo kiosk: Alt+F4 ou Ctrl+W

❓ FAQ

Posso usar Edge / Firefox?

Edge funciona igual ao Chrome: msedge.exe --kiosk-printing. Firefox não suporta impressão silenciosa via flag — fica no Chrome ou Edge.

Funciona com qualquer impressora?

Sim, qualquer impressora reconhecida pelo Windows. Térmica USB (Elgin, Bematech, Epson, Daruma) ou rede.

E se eu tiver 2 impressoras (balcão + cozinha)?

Por enquanto sai tudo na impressora padrão. Pra ter impressora dedicada da cozinha, precisamos integrar via servidor de impressão (Print Server) — abre um chamado pra Belinha.
