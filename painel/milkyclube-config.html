🪙 Economia de MilkyCoins moedas

Define quantas moedas o cliente ganha. As moedas são progresso, não dinheiro — usadas pra desbloquear prêmios dentro do MilkyClube.

🎁 Moedas-bônus de cadastro Cliente novo recebe X moedas pra começar engajado.
